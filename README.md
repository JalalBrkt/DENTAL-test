# HealthFlow Web Export (Flat Structure)

This folder contains a fully functional, static version of the HealthFlow Clinic management system with all files grouped together in a single folder (no sub-directories).

## How to host on GitHub Pages

1. **Option A: The `docs` folder method**
   - Rename this folder to `docs`.
   - Push your project to a GitHub repository.
   - In GitHub Settings > Pages, select your branch and set the folder to `/docs`.

2. **Option B: Dedicated Branch**
   - Push the contents of this folder to a branch named `gh-pages`.

## Configuration Note
Built with relative paths (`base: './'`) and a flat assets structure (`assetsDir: ''`) for maximum compatibility.
